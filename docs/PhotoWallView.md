# `<PhotoWallView />`

> 组件路径：`/App/Components/Common/Widgets/PhotoWall/PhotoWallView.js`

## Methods

### 保存图片

```js
onSaveImagePress();
```

### 长按图片弹出保存图片的弹框

```js
handleLongPress();
```

### 删除照片

```js
onDeletePress();
```

### 切换照片墙,记录当前显示的下标

```js
onAlbumViewChange(newIndex);
```

|          | 类型 | 描述 |
| -------- | ---- | ---- |
| newIndex | -    | -    |

refreshPhotoWall(newPhotoWall);
